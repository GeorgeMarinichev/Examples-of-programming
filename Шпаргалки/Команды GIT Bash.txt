Команды GIT Bash
**Работа с файлами и каталогами (Unix-подобные команды):**
`ls` — показать содержимое текущей директории (например, `ls -la`);  
`cd <путь>` — перейти в указанный каталог (например, `cd my_project`);  
`pwd` — показать текущий каталог;  
`mkdir <имя>` — создать новую папку (например, `mkdir new_folder`);  
`rm <файл>` — удалить файл (например, `rm file.txt`);  
`rm -r <папка>` — удалить папку рекурсивно (например, `rm -r old_folder`);  
`cp <источник> <назначение>` — копировать файл или папку (например, `cp file.txt backup.txt`);  
`mv <источник> <назначение>` — переместить или переименовать файл/папку (например, `mv oldname.txt newname.txt`);  
`cat <файл>` — вывести содержимое файла (например, `cat README.md`);  
`touch <файл>` — создать пустой файл или обновить время модификации (например, `touch newfile.txt`).

---
**Основные команды Git:**
`git init` — инициализировать новый Git-репозиторий;  
`git clone <url>` — склонировать удалённый репозиторий (например, `git clone https://github.com/user/repo.git`);  
`git status` — показать состояние репозитория;  
`git add <файл/папка>` — добавить файл(ы) в индекс (например, `git add .`);  
`git commit -m "сообщение"` — сделать коммит с сообщением (например, `git commit -m "Initial commit"`);  
`git log` — показать историю коммитов;  
`git branch` — показать список веток;  
`git branch <название>` — создать новую ветку (например, `git branch feature`);  
`git checkout <ветка>` — переключиться на другую ветку (например, `git checkout master`);  
`git checkout -b <ветка>` — создать и переключиться на новую ветку (например, `git checkout -b feature`);  
`git merge <ветка>` — слить указанную ветку в текущую (например, `git merge feature`);  
`git pull` — забрать изменения с удалённого репозитория и слить их (например, `git pull origin master`);  
`git push` — отправить локальные коммиты в удалённый репозиторий (например, `git push origin master`);  
`git remote -v` — показать список удалённых репозиториев.

---
**Полезные команды оболочки Git Bash:**
`clear` — очистить экран терминала;  
`history` — показать историю введённых команд;  
`exit` — выйти из терминала.
